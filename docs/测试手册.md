## 🐛 故障排查

### 常见问题

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 天气查询失败 | 代理问题 | 检查代理配置是否正确 |
| 网页摘要超时 | 网络问题 | 换个网址试试 |
| 知识库无结果 | 知识库为空 | 先上传文档到知识库 |
| 页面无响应 | 服务崩溃 | 重启服务 |
| API 调用失败 | API Key 问题 | 检查 .env 中的 KIMI_API_KEY |

### 重启服务命令

**Docker 方式：**
```bash
docker restart jingjing-agent
```

**本地方式：**
```bash
pkill -f "streamlit run"
./start.sh
```

### 查看日志

**Docker：**
```bash
docker logs -f jingjing-agent
```

**本地：**
```bash
tail -f streamlit.log
```
