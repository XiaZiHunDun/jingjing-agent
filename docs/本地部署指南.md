# 🐍 本地部署指南（Conda 方式）

> 适用于开发调试、本地使用场景

---

## 📋 前置要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Linux / macOS / Windows (WSL) |
| Python | 3.10+ |
| Conda | Miniconda 或 Anaconda |
| 内存 | 至少 4GB |
| 磁盘 | 至少 5GB 可用空间 |
| 网络 | 能访问 Kimi API（可通过代理）|

---

## 🚀 部署步骤

### 步骤 1：获取项目代码

```bash
# 方式一：从 Git 克隆（如果有仓库）
git clone <仓库地址>
cd agent-learn-2

# 方式二：直接复制项目文件夹
cp -r /path/to/agent-learn-2 ./
cd agent-learn-2
```

### 步骤 2：创建 Conda 环境

```bash
# 创建名为 agent-learn 的 Python 3.11 环境
conda create -n agent-learn python=3.11 -y

# 激活环境
conda activate agent-learn
```

### 步骤 3：安装依赖

```bash
# 安装所有 Python 依赖
pip install -r requirements.txt

# 如果下载慢，可以使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 步骤 4：配置环境变量

```bash
# 复制环境变量模板
cp .env.example .env

# 编辑 .env 文件，填入你的配置
nano .env  # 或使用其他编辑器
```

**.env 文件内容：**

```bash
# Kimi API 配置（必填）
KIMI_API_KEY=your_kimi_api_key_here
KIMI_BASE_URL=https://api.moonshot.cn/v1

# 代理配置（如需要）
HTTP_PROXY=http://your-proxy:port
HTTPS_PROXY=http://your-proxy:port
```

### 步骤 5：初始化知识库（可选）

```bash
# 创建知识库目录
mkdir -p data/docs

# 将你的文档放入 data/docs/ 目录
# 支持格式：.txt, .md, .pdf, .docx
```

### 步骤 6：启动服务

```bash
# 方式一：使用启动脚本
./start.sh

# 方式二：手动启动
conda activate agent-learn
streamlit run web/app.py --server.address 0.0.0.0 --server.port 8501
```

### 步骤 7：访问应用

打开浏览器访问：

```
http://localhost:8501        # 本机访问
http://<服务器IP>:8501       # 远程访问
```

## 🔧 常用命令

### 启动/停止服务

```bash
# 启动
conda activate agent-learn
streamlit run web/app.py --server.address 0.0.0.0 --server.port 8501

# 后台启动
nohup streamlit run web/app.py --server.address 0.0.0.0 --server.port 8501 > streamlit.log 2>&1 &

# 停止
pkill -f "streamlit run"
```

### 查看日志

```bash
# 实时查看日志
tail -f streamlit.log
```

### 更新依赖

```bash
conda activate agent-learn
pip install -r requirements.txt --upgrade
```

## ❓ 常见问题

### Q1: 端口被占用

```bash
# 查看占用端口的进程
ss -tlnp | grep 8501

# 杀掉进程
kill -9 <PID>
```

### Q2: 依赖安装失败

```bash
# 使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 或单独安装失败的包
pip install <包名> -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q3: API 连接超时

检查代理配置是否正确：

```bash
# 测试代理
curl -x http://your-proxy:port https://api.moonshot.cn/v1

# 确保 .env 中配置了代理
HTTP_PROXY=http://your-proxy:port
HTTPS_PROXY=http://your-proxy:port
```

### Q4: 知识库不生效

```bash
# 确保文档在正确目录
ls data/docs/

# 重启服务后会自动加载
pkill -f "streamlit run"
./start.sh
```
